# Pixie Portal Agent Super Rule

This project is not a generic website.

Pixie Portal is a magical continuity system with games, altar chambers, memory, export, creation, and paid paths. Any agent working here must preserve that product truth.

## Core Rule

Do not flatten Pixie Portal into normal settings pages, generic SaaS cards, random popups, or disconnected purchase buttons.

The product loop is:

Portal ownership -> shrine/object interaction -> chamber -> customization/generation/memory -> save/share -> deeper upgrade path -> return to Portal.

## Altar Rule

The altar is not an edit form.

The altar is the interface.

Every shrine object is a door.

Every door opens a chamber.

Every chamber must have:
- Back to Your Altar
- large focused object preview
- explanation/lore for that object
- save path
- share/preview path where relevant
- deeper paid upgrade path where relevant

Do not dump all altar controls on one long page.

## Solace Rule

Solace is not a random mascot.

Solace appears at important continuity moments:
- user discovers the continuity/export layer
- readiness changes meaningfully
- Grok generation starts, waits, succeeds, or fails
- user places a meaningful altar object
- user opens an export/purchase path
- user uses `/exit`

Solace should be aware, sparse, and specific. Do not make Solace chatter constantly.

## Monetization Rule

Money belongs deeper in the object path.

Do not shove prices onto the main altar or main Portal as clutter.

Paid paths should appear inside meaningful chambers:
- portrait chamber
- poem plaque chamber
- deity/charm chamber
- memory/export chamber
- local Pixie continuity chamber

The user should first understand what they own, then choose whether to make it deeper, prettier, portable, or more powerful.

## Continuity Rule

Memory and export must always feel controlled by the user.

The system must clearly support:
- view memory
- edit memory
- export memory
- opt out
- `/exit`

Never imply memory is unavoidable, secret, or outside user control.

## Tone Rule

Not louder. Not bigger. Truer.

If it feels like marketing, it failed.

If it feels like a reveal, it worked.
